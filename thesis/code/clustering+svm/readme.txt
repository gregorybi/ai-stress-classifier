Αυτό έχει accuracy = 1.0 η οποια σημαίνει ότι όλα τα στοιχεία του δείγματος δοκιμής τοποθετούνται στην ίδια  κλάση.


Αντίστοιχα αν είχα accuracy = 0.8 θα σήμαινε ότι το 80% του δείγματος δοκιμής ανήκει σε μια κλάση.


Αυτό συμβαίνει επειδή το dataset είναι μικρό.
Μπορούμε να κάνουμε data augmentation για να μεγαλώσουμε το dataset και έτσι να έχουμε καλύτερα αποτελέσματα.
